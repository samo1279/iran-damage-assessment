# Screenshots

To add screenshots:

1. Open the app at: https://formula-sleeve-mixing-reconstruction.trycloudflare.com
2. Take screenshots of:
   - Main dashboard view
   - Satellite comparison panel
   - Strike analysis panel
   - Mobile view

3. Save them as:
   - `dashboard.png`
   - `comparison.png`
   - `strikes.png`
   - `mobile.png`

4. Place them in this `docs/screenshots/` folder

5. Commit and push:
   ```bash
   git add docs/
   git commit -m "Add screenshots"
   git push
   ```
